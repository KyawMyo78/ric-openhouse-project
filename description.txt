PROJECT TITLE:
Rock Paper Scissors AI Emotion Detector

SHORT DESCRIPTION:
An intelligent Rock Paper Scissors game that combines computer vision, AI emotion detection, and real-time hand gesture recognition to create an interactive gaming experience that responds to your facial expressions.

DETAILED DESCRIPTION:
This innovative project transforms the classic Rock Paper Scissors game into an AI-powered interactive experience. Using advanced computer vision techniques, the application detects hand gestures in real-time to recognize Rock, Paper, and Scissors moves while simultaneously analyzing facial expressions to determine the player's emotional state. The game features adaptive UI scaling that automatically adjusts to different camera resolutions, multi-camera support with interactive selection, and emotion-based responses that change based on whether you're happy, sad, surprised, sleepy, or neutral. The system includes a professional countdown timer, landmark visualization toggle for debugging, and cross-platform audio feedback. Built with MediaPipe for landmark detection and OpenCV for computer vision processing, this project demonstrates practical applications of AI in gaming and human-computer interaction.
